# Bank-Management-System-C
A Bank Management System built in C using file handling and structured programming.
Bank Management System (C Language)

📌 Description

This is a console-based Bank Management System developed in C language using file handling and structured programming. It allows users to perform basic banking operations such as account creation, deposit, withdrawal, and transaction tracking.

🚀 Features

- Create, update, and delete accounts
- Deposit and withdraw money
- Balance inquiry
- Transaction history
- Login system with PIN
- File handling for data storage

🛠️ Technologies Used

- C Programming Language
- File Handling (fopen, fread, fwrite)

▶️ How to Run

1. Open the project in Dev-C++
2. Compile the code
3. Run the program

📂 Files

- bank_management_system.c
- accounts.dat
- transactions.dat
